# Bitmap Converter

Convert images to bitmap patterns using halftone, dithering, and threshold methods.

## Quick Start

```bash
# Install dependencies
npm install

# Run development server
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

## Build

```bash
npm run build
npm start
```

## Tech Stack

- Next.js 16 + React 19
- TypeScript
- CSS Modules
- Canvas API for image processing

## Features

- Multiple bitmap conversion methods (halftone, threshold, pattern/diffusion dither)
- 6 halftone shapes (line, round, diamond, ellipse, square, cross)
- Background removal
- Real-time preview
- Export at 1×-4× resolution
